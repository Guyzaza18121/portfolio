01 — เกี่ยวกับผม

สร้างซอฟต์แวร์ที่แก้ปัญหาจริง

ผมเป็นนักพัฒนาที่หลงใหลในการเปลี่ยนปัญหาที่ซับซ้อนให้กลายเป็นระบบที่ เรียบง่ายและใช้งานได้จริง ตั้งแต่ระบบหลังบ้านสำหรับร้านค้า ไปจนถึง Dashboard ควบคุมอุปกรณ์ IoT ในโรงงานผ่าน MQTT และ Node-RED

ผมทำงานครบทั้ง Frontend, Backend และการเชื่อมต่ออุปกรณ์ โดยให้ความสำคัญกับประสบการณ์ผู้ใช้ ความเสถียร และการ Deploy ที่พร้อมใช้งานบน Server จริง

Jewelry Admin

  • สร้าง PDF ภาษาไทยด้วย pdf-lib
